# Task Manager Web App

Данный проект представляет собой простое веб-приложение для управления задачами.
Пользователь может добавлять, редактировать и удалять задачи, а также отмечать их как выполненные.
Приложение предназначено для организации личных задач.
Разработка проекта ведётся с использованием Agile и Scrum подходов.
В рамках лабораторной работы смоделирован один Scrum-спринт с использованием GitHub Issues.

## Sprint Review

В ходе спринта были завершены задачи по регистрации пользователя и созданию задач.
Реализован базовый функционал приложения.
Задача по отметке задач как выполненных не была завершена в рамках данного спринта.

## Sprint Retrospective

Что прошло хорошо:
- чёткая формулировка задач
- понятные критерии готовности

Что пошло не так:
- неточная оценка времени выполнения задач

Что можно улучшить:
- более точное планирование спринта
- улучшение оценки сложности задач
