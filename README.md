# Task Manager Web App

Данный проект представляет собой простое веб-приложение для управления задачами.
Пользователь может добавлять, редактировать и удалять задачи, а также отмечать их как выполненные.
Приложение предназначено для организации личных задач.
Разработка проекта ведётся с использованием Agile и Scrum подходов.
В рамках лабораторной работы смоделирован один Scrum-спринт с использованием GitHub Issues.
